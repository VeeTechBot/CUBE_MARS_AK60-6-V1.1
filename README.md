# CUBE_MARS_AK60-6-V1.1
This is a repository for storing the files of AK60-6 V 1.1 which can be used to flash using ST-Link, or updating the bootloader settings.

There are three types of files in this repository:

1) .bin files: These files are flashed using the bootloader mode. There are two types of .bin files, one file for MIT and the other for Servo mode. Chsnge your motor to bootloader mode and then download the following files.
2) .AppParams and McParams: These files are specific for a particulor motor and can imported uing the "Import Setting" Options. Do not forget to write the prametres using the "Write Parametres" option
3) .hex files: If you are unable to switch the mode of the motor you have to flash the motor from connecting ST-Link to PCB of the motor.
